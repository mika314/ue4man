# NAME
**SetNotifyRigidBodyCollision** - sets the checkbox Simulation Generates Hit Events.

# DESCRIPTION


Note: does not work for skeletal meshes :(

# Code sample

```c++
  mesh->SetNotifyRigidBodyCollision(true);
```
